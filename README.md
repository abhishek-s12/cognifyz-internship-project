# Cognifyz Full Stack Development Internship

Internship tasks completed for Cognifyz Technologies.

## Tech Stack
- Node.js + Express
- EJS Templating
- MongoDB Atlas + Mongoose
- Bootstrap 5
- JWT Authentication
- bcryptjs
- OpenWeatherMap API

## Levels

### Level 1 — Beginner
- Task 1: HTML Structure + Express Server + EJS Rendering
- Task 2: Client-side + Server-side Form Validation

### Level 2 — Intermediate
- Task 3: Advanced CSS + Bootstrap 5 + Responsive Design
- Task 4: Password Strength + Dynamic DOM + SPA Routing

### Level 3 — Advanced
- Task 5: RESTful API + CRUD Operations + fetch()
- Task 6: MongoDB Integration + JWT Authentication

### Level 4 — Expert
- Task 7: External API Integration (OpenWeather) + Rate Limiting
- Task 8: Middleware (Morgan) + Node-Cache + Background Jobs

## Setup

1. Clone the repo
2. Go into any level folder
3. Run `npm install`
4. Create `.env` with your credentials
5. Run `node server.js`
